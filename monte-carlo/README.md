# Monte Carlo Simulation – Geometric Brownian Motion (GBM)

This project simulates stock price paths using Geometric Brownian Motion (GBM) and applies Monte Carlo methods for financial analysis.

## Model
GBM assumes stock prices follow:

dS = μS dt + σS dW

Where:
- μ = expected return (drift)
- σ = volatility
- dW = Brownian motion (random shock)

## What this project does
- Simulates multiple stock price paths
- Visualizes simulated trajectories
- Shows distribution of terminal prices
- Estimates return and volatility
- Prices a European call option using Monte Carlo
